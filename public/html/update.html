HeadTab 2.1

HeadTab has just been updated to the next release: 2.1
This release packs the latest features, integrations and improvements

Features
- Added integrations, HeadTab can now integrate seamlessly with some services

Integrations
- Spotify integration, display playing songs in the info dashboard
- Spotify lyrics integration, display lyrics of the current song when turned on in Spotify
- Youtube integration, display playing videos in the info dashboard

Improvements
- Battery status, display the battery on the bottom left
- Settings now makes the different types input fields more clear
